# Next Letta features to implement

Research goal: identify Letta Desktop features that are publicly visible in Letta’s recent product updates, but have not yet been explored in our backlog. Schedules are already handled, so they are excluded here.

## 1) Skills management and installation

**What it is**
Letta agents can gain new abilities through "skills". Public examples include browser use (Playwright), ticket management (Linear), and image-related workflows.

**How it works**
Skills act like capability packs that extend an agent’s toolset and behavior. Letta has shown that agents can install skills themselves when given a skill link, which suggests a lightweight sharing/install flow rather than a heavy plugin marketplace.

**Why it’s valuable**
This makes agents more adaptable to different jobs and teams. Instead of hard-coding every behavior into the app, users can add or swap capabilities as needs change.

**Research sources**
- Letta LinkedIn post on skills: browser use, ticket management, image generation, and self-installing skills
- Letta homepage / product summaries describing agents as teachable and customizable

---

## 2) Channel integrations for Slack, Telegram, Discord, and custom channels

**What it is**
Letta agents can work across team communication channels, not just inside the desktop app. Public announcements specifically mention Slack, Telegram, Discord, and custom channels.

**How it works**
A single persistent agent can be connected to multiple surfaces. Team members interact with the same agent where they already collaborate, and the agent can respond with text and multimodal content. Letta has also described agents running on any machine and being reachable through these channels.

**Why it’s valuable**
This is a major distribution feature: it brings the same stateful agent to the team’s existing workflows. It also makes the agent feel shared and collaborative instead of trapped in a single client.

**Research sources**
- Letta LinkedIn post announcing Slack/Telegram/Discord/custom channels
- Letta office-hours post on connecting agents to Telegram and Slack
- Letta Slack update noting cross-machine interface support and multimodal content

---

## 3) Built-in image generation and image modification

**What it is**
Letta agents now support built-in image generation, including image modification. Letta has mentioned support from Black Forest Labs, Google Gemini, and OpenAI.

**How it works**
Image generation is exposed as an agent capability, so users can ask the agent to create or edit images directly. Letta also notes that images can be delivered through channels, which extends the feature beyond the desktop UI.

**Why it’s valuable**
This turns Letta from a text-first assistant into a multimodal creative agent. It is especially useful for marketing, design exploration, social content, and other workflows where visual output matters.

**Research sources**
- Letta LinkedIn post announcing built-in image generation
- Letta LinkedIn post noting image generation support through channels

---

## Recommendation

If we want the next three research-driven feature bets for Letta Desktop, I’d prioritize:
1. Skills management and installation
2. Channel integrations
3. Built-in image generation

These three features are distinct, user-visible, and align well with Letta’s core positioning around persistent, stateful agents that work where people already work.
